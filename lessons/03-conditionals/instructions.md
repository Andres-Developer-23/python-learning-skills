# 🔀 Lección 3: Estructuras de Control

## 🎯 Objetivo
Aprender a tomar decisiones en tu código usando condicionales.

## 📝 Tu tarea
Crea un programa que:
1. Pida la edad del usuario
2. Clasifique la edad en categorías:
   - Menor de 13: "Niño"
   - 13 a 17: "Adolescente"
   - 18 a 64: "Adulto"
   - 65 o más: "Adulto mayor"
3. Muestre un mensaje según la categoría

## 🔧 Requisitos
- Usa `input()` para obtener la edad
- Usa `if`, `elif` y `else` para las condiciones
- Usa `int()` para convertir la entrada a número
- El programa debe ejecutarse sin errores

## 💡 Ejemplo

```
¿Cuántos años tienes? 25
Eres Adulto. ¡Tienes derecho a votar!
```

## 💡 Ejemplo con errores

```
¿Cuántos años tienes? diez
Por favor, ingresa un número válido.
```

## 🚀 ¡Manos a la obra!
Escribe tu código en un archivo llamado `solucion.py`
