# 📊 Lección 2: Variables y Tipos de Datos

## 🎯 Objetivo
Crear variables de diferentes tipos.

## 📝 Tu tarea
Crea un programa que:
1. Defina estas variables:
   - nombre (texto)
   - edad (número)
   - altura (número con decimal)
   - hobbies (lista)
2. Muestre toda la información

## 🔧 Requisitos
- Usa diferentes tipos de datos
- Usa f-strings para mostrar mensajes
- El programa debe ejecutarse sin errores

## 💡 Ejemplo

- Nombre: María
- Edad: 25 años
- Altura: 1.65 metros
- Hobbies: [música, leer, viajar]
