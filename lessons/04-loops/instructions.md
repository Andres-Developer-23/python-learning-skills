# 🔄 Lección 4: Bucles y Listas

## 🎯 Objetivo
Aprender a repetir acciones y manejar colecciones de datos.

## 📝 Tu tarea
Crea un programa que:
1. Pida al usuario 3 hobbies usando un bucle
2. Guarde los hobbies en una lista
3. Muestre todos los hobbies ingresados
4. Cuente cuántos hobbies fueron ingresados

## 🔧 Requisitos
- Usa un bucle `for` o `while` para repetir la entrada
- Usa una lista para guardar los hobbies
- Usa `.append()` para agregar elementos
- Usa `len()` para contar elementos
- El programa debe ejecutarse sin errores

## 💡 Ejemplo

```
Ingresa tu hobby 1: Leer
Ingresa tu hobby 2: Correr
Ingresa tu hobby 3: Cocinar

Tus hobbies son:
1. Leer
2. Correr
3. Cocinar

Tienes 3 hobbies.
```

## 🚀 ¡Manos a la obra!
Escribe tu código en un archivo llamado `solucion.py`
