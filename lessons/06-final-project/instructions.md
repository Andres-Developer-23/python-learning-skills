# 🎯 Lección 6: Proyecto Final

## 🎯 Objetivo
Demostrar todo lo aprendido creando un programa completo.

## 📝 Tu tarea
Crea un **sistema simple de registro de estudiantes** que:
1. Pida al usuario cuántos estudiantes va a registrar
2. Use un bucle para pedir los datos de cada estudiante:
   - Nombre
   - Nota (0-100)
3. Use condicionales para determinar la cualificación:
   - 90-100: "Sobresaliente"
   - 70-89: "Notable"
   - 50-69: "Aprobado"
   - 0-49: "Suspenso"
4. Guarde los datos en una lista de diccionarios
5. Use una función para calcular el promedio del grupo
6. Muestre un resumen final con todos los estudiantes y sus cualificaciones

## 🔧 Requisitos
- Usa `input()` para obtener datos
- Usa `int()` o `float()` para convertir números
- Usa `if/elif/else` para cualificaciones
- Usa `for` o `while` para iterar
- Usa listas y diccionarios
- Usa al menos 2 funciones
- Usa `f-strings` para formatear mensajes
- El programa debe ejecutarse sin errores

## 💡 Ejemplo

```
¿Cuántos estudiantes vas a registrar? 2

Estudiante 1:
¿Nombre? Ana
¿Nota? 95

Estudiante 2:
¿Nombre? Carlos
¿Nota? 72

=== RESUMEN DEL GRUPO ===
Ana - Nota: 95 - Sobresaliente
Carlos - Nota: 72 - Notable

Promedio del grupo: 83.5
```

## 🚀 ¡Manos a la obra!
Escribe tu código en un archivo llamado `solucion.py`
