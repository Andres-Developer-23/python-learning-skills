# ⚡ Lección 5: Funciones

## 🎯 Objetivo
Aprender a crear y usar funciones para organizar tu código.

## 📝 Tu tarea
Crea un programa que:
1. Defina una función `calcular_promedio(numeros)` que reciba una lista de números y devuelva el promedio
2. Defina una función `es_par(numero)` que devuelva True si el número es par, False si es impar
3. Use ambas funciones con datos de ejemplo
4. Muestre los resultados

## 🔧 Requisitos
- Define al menos 2 funciones con `def`
- Usa parámetros y valores de retorno
- Usa `return` para devolver resultados
- El programa debe ejecutarse sin errores

## 💡 Ejemplo

```
Números: [10, 20, 30, 40, 50]
Promedio: 30.0

¿15 es par? False
¿20 es par? True
```

## 🚀 ¡Manos a la obra!
Escribe tu código en un archivo llamado `solucion.py`
