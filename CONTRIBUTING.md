# 🤝 Contribuir al Curso

¡Gracias por tu interés en aprender Python! Este curso está diseñado para ser completamente interactivo usando Issues y Pull Requests de GitHub.

## 🎯 Dos formas de participar

### Método 1: Issues (para principiantes)
1. Abre un Issue usando la plantilla de la lección
2. Pega tu código Python en el bloque ` ```python `
3. El workflow revisará automáticamente tu código
4. Recibirás feedback en el mismo Issue

### Método 2: Pull Requests (para avanzados)
1. Haz fork del repositorio
2. Clona tu fork: `git clone https://github.com/tu-usuario/python-learning-skills.git`
3. Crea una rama: `git checkout -b leccion-01`
4. Escribe tu código en `lessons/01-intro/solucion.py`
5. Commit y push: `git push origin leccion-01`
6. Abre un Pull Request

## 📋 Requisitos
- Tu código debe ser Python 3.10+
- Sigue las instrucciones de cada lección
- Usa los nombres de variables indicados
- Asegúrate de que tu código se ejecute sin errores

## 🐛 Reportar problemas
Si encuentras errores en las lecciones o el sistema, abre un Issue con la etiqueta `bug`.

## 💡 Sugerencias
¿Tienes ideas para mejorar el curso? Abre un Issue con la etiqueta `enhancement`.
